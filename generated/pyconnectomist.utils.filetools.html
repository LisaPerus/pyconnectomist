pyconnectomist.utils.filetools.gz_compress(file_to_compress, clean=True)[source]¶

Compress a file with gzip, the output path is the same but with ”.gz” extension.

The function raises ConnectomistBadFileError if the input file does not exist or if the output compressed file is not created.

Parameters:

file_to_compress: str

the file to compress.

clean: bool (optional, default True)

If ‘clean’ is True, the input file is deleted, to keep only the compressed version.
Returns:

gz_file: str

the compressed output file.

pyconnectomist.utils.filetools.ptk_concatenate_volumes(path_inputs, path_output, axis='t')[source]¶

Function that wraps the PtkCat command line tool from Connectomist, with only the basic arguments. It allows concatenating volumes. In particular to concatenante T2 and DW volumes in one file at the end of the preprocessing.

Parameters:

path_inputs: list of str

paths to input volumes.

path_output: str

path to output volume.

axis: str (optional, default ‘t’)

axis along which the concatenation is done.
Returns:

path_output: str

path to output gis volume.
Raises:

ConnectomistRuntimeError: If call to PtkCat failed.

pyconnectomist.utils.filetools.ptk_split_t2_and_diffusion(t2_dw_input, t2_output, dw_output)[source]¶

Function meant to split a Gis file containing a T2 volume (first volume) and diffusion-weigthed volumes (the other volumes) in 2 Gis files. The separation is done using the PtkSubVolume command line tool from Connectomist.

Parameters:

t2_dw_input: str

path to input volume.

t2_output: str

path to output T2 volume.

dw_output: str

path to output diffusion-weighted volumes.
Returns:

t2_output: str

the t2 diffusion Gis volume.

dw_output: str

the diffusion weighted Gis volume.
Raises:

ConnectomistRuntimeError: If call to PtkSubVolume failed.
