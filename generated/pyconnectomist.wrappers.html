classmethod create_parameter_file(algorithm, parameters_dict, outdir)[source]¶

Writes the ‘.py’ file Connectomist uses when working in command line.

Parameters:

algorithm: str

name of Connectomist’s tab.

parameters_dict: dict

parameter values for the tab.

outdir: str

path to directory where to write the parameter file. If not existing the directory is created.
Returns:

parameter_file: str

path to the created parameter file for the tab in ui: executable python file to set the connectomist tab input parameters.
